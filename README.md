#Exercícios de Armazenamento e Gestão de Dados

Este repositório armazena as resoluções dos exercícios propostos durante as disciplinas de Análise e Gestão de Dados e Banco de Dados II. As implementações estão em SQL (MySQL) e os arquivos contêm os scripts de cada resolução.

##Conteúdo do Repositório

Os exercícios estão organizados por revisão, cada uma abordando um conjunto de comandos e conceitos distintos.

Revisão 01 - DDL (Data Definition Language): Exercícios focados na definição e estruturação de bancos de dados, utilizando comandos como CREATE TABLE, ALTER TABLE e DROP TABLE. Os scripts estão separados por cenário: universitário, aéreo e biblioteca.

Revisão 02 - DML (Data Manipulation Language): Exercícios focados na manipulação de dados sobre os mesmos cenários da Revisão 01, utilizando comandos INSERT, UPDATE e DELETE.

Revisão 03 - SQL Simples: Exercícios de consulta SELECT sobre um banco de dados de filmes, abordando filtros com WHERE, ordenação com ORDER BY e agrupamentos com GROUP BY e HAVING.

Revisão 05 - JOIN (Junção): Exercícios de consulta envolvendo junção de tabelas, utilizando INNER JOIN, LEFT JOIN e RIGHT JOIN sobre o banco de dados Sakila.
