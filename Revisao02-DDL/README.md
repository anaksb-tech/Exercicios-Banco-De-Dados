Exercícios focados na manipulação de dados sobre os mesmos bancos de dados
criados na Revisão 01, usando comandos INSERT, UPDATE e DELETE.

Os scripts estão separados em três arquivos, cada um correspondendo a um
cenário distinto:

universitario.sql

    Inserção dos dados nas tabelas ALUNO, DISCIPLINA, TURMA, PRE_REQUISITO
    e HISTORICO_ESCOLAR conforme os dados fornecidos no enunciado.

aereo.sql

    Inserção de 5 registros em cada tabela do banco de dados de voos.
    Inclui também no mínimo 3 alterações (UPDATE) e 3 exclusões (DELETE)
    de dados distribuídos entre as tabelas.

biblioteca.sql

    Inserção de 5 registros em cada tabela do banco de dados de biblioteca.
    Inclui também no mínimo 3 alterações (UPDATE) e 3 exclusões (DELETE)
    de dados distribuídos entre as tabelas.

Observação: as tabelas LIVRO_AUTOR e UNIDADE_BIBLIOTECA foram removidas
na Revisão 01. Os INSERTs referentes a elas foram mantidos no arquivo com
comentários explicativos, mas não devem ser executados após o DROP.
