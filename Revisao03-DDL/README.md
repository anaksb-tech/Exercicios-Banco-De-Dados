Exercícios de consulta sobre o banco de dados "cinema_sql_simples", que
contém uma única tabela chamada "filmes". Os exercícios cobrem os principais
recursos de consulta SQL:

- Filtros simples com WHERE (comparação, AND, BETWEEN)
- Ordenação de resultados com ORDER BY (ASC e DESC)
- Agrupamento de dados com GROUP BY
- Filtros sobre grupos com HAVING
- Funções de agregação: COUNT, SUM, AVG

As consultas estão no arquivo cinema.sql, organizadas na ordem dos itens
a até z conforme o enunciado.
